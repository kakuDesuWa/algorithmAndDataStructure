# 问题描述

一个环上有10个点，编号为0-9，从0点出发，每步可以顺时针到下一个点，也可以逆时针到上一个点，求：经过n步又回到0点有多少种不同的走法

e.g: 

            0
        9       1
    8               2
                    
    7               3
                    
        6       4
            5

n步可左可右, 例如:

1. n = 1, 则从0出发, 可到1/9, 但都不能回到原点0, 故返回0
2. n = 2, 共有 0-1-2, 0-1-0, 0-9-8, 0-9-0, 四种, 其中2条能返回0, 故返回2
